# GuitarTuner

# Gereklilikler
* Python Versiyon 2.7.16
* numpy Versiyon 1.21.2
* scipy Versiyon 1.1.0
* sounddevice Versiyon 0.4.4


# Dependencies
## realTimeNoteDedect
* pip install numpy
* pip install scipy
* pip install sounddevice 
* sudo apt-get install libportaudio2

## voiceRecorder
* pip3 install sounddevice
* pip3 install wavio
* pip3 install scipy
 
## wavToText
* pip install numpy
* pip install scipy
* pip install matplotlib
